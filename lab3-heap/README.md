## HeapSort  

**Опис:**  
Сортування купою (Heap Sort) ґрунтується на побудові **бінарної купи** — повного дерева, у якому кожен батьківський елемент більший за свої дочірні.  
Після побудови максимальної купи найбільший елемент переміщується в кінець масиву, а процес повторюється для решти елементів.  

**Характеристики:**  
- **Складність:**  
  - Кращий випадок: O(n log n)  
  - Середній випадок: O(n log n)  
  - Гірший випадок: O(n log n)  
- **Пам’ять:** O(1)  
- **Стабільність:** ❌ Ні  
- Підходить для великих наборів даних, оскільки не потребує додаткової пам’яті та має стабільну продуктивність.
